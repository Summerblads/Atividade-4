Relatório de Estágio

1. Introdução

O estágio é uma maneira de qualificar o discente que está exercendo suas funções, e necessita de um maior aprofundamento em sua área. A empresa na qual exerci o meu estágio, é a "Holofotes Computadores", uma empresa especializada em manutenção e reparos de máquinas de processamento de dados, isto é, computadores.

1.1. Objetivos

1.1.1. Objetivo Geral

Aplicar os conhecimentos adquiridos durante o curso em um ambiente profissional real.

1.1.2. Objetivos Específicos

Aprimorar habilidades técnicas, desenvolver soluções práticas e adquirir experiência com metodologias ágeis.

2. Atividades Realizadas

2.1. Desenvolvimento de um sistema web

Participei do desenvolvimento de um sistema web para controle de estoque utilizando HTML, CSS, JavaScript e PHP.

2.2. Integração com API externa

Implementei funcionalidades que integravam o sistema com uma API REST para obter dados de fornecedores.

3. Conclusão

O estágio proporcionou aprendizado prático e crescimento profissional, contribuindo significativamente para minha formação.
